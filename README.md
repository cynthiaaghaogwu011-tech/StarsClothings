# StarsClothings

StarsClothings is a modern fashion e-commerce website built with HTML, Tailwind CSS, and JavaScript. It provides a clean and responsive shopping experience where users can browse products, view product details, add items to their cart, and complete checkout.

## Features

- Responsive design
- Home page with featured collections
- Shop page
- Product details page
- Shopping cart
- Checkout page
- Search functionality
- Local Storage cart persistence
- Mobile-friendly navigation

## Technologies Used

- HTML5
- Tailwind CSS
- JavaScript
- Local Storage API

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/cynthiaaghaogwu011-tech/StarsClothings.git
```

2. Open the project folder.

3. Open `index.html` in your browser or use the VS Code Live Server extension.
